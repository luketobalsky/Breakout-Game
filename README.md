# Breakout-Game
CMSC495 Capstone Project, Breakout Game in Python

Description:
A classic Breakout-style arcade game built in Python using Pygame. 
The player control a paddle to bounce a ball and break bricks to score points.

Core Features:
1. Paddle movement
2. Ball physics and collision detection
3. Brick grid
4. Score tracking
5. Player lives tracking
6. Game restart


Requirements:
1. Python 3.11+ 
2. Pygame


Installation:
git clone "repo-url".git

Run game:
1. Via code editor:
   a. run the main class
2. Via command line:
   a. Open command line/terminal
   b. Cd into the Breakout-Game directory
   c. Once inside correct directory run: python3 -m main or python main
