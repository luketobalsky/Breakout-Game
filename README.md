# Breakout-Game
CMSC495 Capstone Project, Breakout Game in Python

Description:
A classic Breakout-style arcade game built in Python using Pygame. 
The player control a paddle to bounce a ball and break bricks to score points.

Core Features:
1. Paddle movement
2. Ball physics and collision detection
3. Brick grid
4. Score tracking
5. Player lives tracking
6. Game restart


Requirements:
1. Python 3.11+ 
2. Pygame


Installation:
git clone "repo-url".git

Run game:
1. Open command line or terminal
2. Cd into the Breakout-Game directory
3. Once inside correct directory run: python3 -m game.main or python game.main
